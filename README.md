# excel2ralf
a python script which is used for transforming register Excel file into ralf file. 

## install dependence
```
pip install -r requirePackage.txt
```

## usage
generate module register ralf file
```
python .\main.py -f .\example\demo_wdt_reg_spec.xlsx
```
